# Colattanini — Laminated Collector Sheet Spec
_Asset spec · MVP physical collector card · Created 2026-06-07_

The small **laminated sheet** the customer keeps and sticks their 3 Colattanini
stickers onto. Completing all 3 slots = eligible for **today's Colattanini reward**.
Manual campaign — see `PRODUCT_MODULES/COLATTAO_COLATTANINI_COLLECTIBLE_CAMPAIGN.md`.

---

## Physical spec

| Property | Value |
|---|---|
| Size | **Credit‑card landscape (≈85 × 54 mm)** or A6 (105 × 148 mm) if more art room is wanted |
| Material | Cardstock, **laminated** (so the same sheet survives 3 visits) |
| Finish | Matte laminate (less glare); rounded corners |
| Bleed / safe area | 3 mm bleed; keep text 4 mm inside the trim |
| Sticker slots | **3 clearly marked circles/squares** sized for the 2 in stickers |

---

## Layout (front)

```
┌───────────────────────────────────────┐
│  COLATTANINI  ·  Colattao Penalty      │   ← header + tiny game/QR mark
│  Collect all 3!  ·  ¡Junta los 3!      │
│                                        │
│   [ slot 1 ]    [ slot 2 ]   [ slot 3 ]│
│  Churro        Coco Beach    Cali       │
│  Chispa        Blitzer       Croissant  │
│  (Street)      (Club)        (Pro)      │
│  Churro Latte  Coco Beach    Cali Sand. │
│                                        │
│  Full set = today's Colattanini reward │
└───────────────────────────────────────┘
```

- Each slot is labeled with **character + keeper level + signature item** so the
  customer knows which keeper to play next.
- A faint outline/silhouette of each character can pre‑print in each empty slot.

## Copy on the sheet (bilingual)

- Header: **COLATTANINI — Colattao Penalty Shootout**
- Call to action — EN: **"Collect all 3!"** · ES: **"¡Junta los 3!"**
- Slot labels:
  - Slot 1: **Churro Chispa** — Street Keeper — *Churro Latte*
  - Slot 2: **Coco Beach Blitzer** — Club Keeper — *Coco Beach*
  - Slot 3: **Cali Croissant Captain** — Pro Keeper — *California Sandwich*
- Footer — EN: **"Complete the set for today's Colattanini reward."**
  ES: **"Completa la colección por la recompensa Colattanini de hoy."**
- Small print — EN: "No app, no sign‑up, nothing to download." ES: "Sin app, sin registro, nada que descargar."

## Back (optional)
- 1‑line "How to play": **Scan → Play a keeper → Show staff → Get your sticker.**
  ES: **Escanea → Juega un portero → Muestra al staff → Llévate tu sticker.**
- Small QR to the Penalty Shootout game (optional convenience copy of the counter QR).

## Guardrails
- Footer must say the reward is **"today's Colattanini reward"** (no specific prize)
  until approved in writing.
- **No customer data fields** on the sheet (no name/phone/email lines). It is a
  keepsake, not a tracking card.
- Original art only; no licensed sports branding (see `GAME_CUSTOMIZATION_PROTOCOL.md`).
