# Colattao Digital Experience Starter Protocol

## Colattao Campaign Protocols

Every Colattao campaign should identify:

- Campaign name.
- Customer action: play, view Digital Menu, scan QR, visit, request update, or share.
- Surface: menu page, game, website concept, signage, sticker, owner flow, or social asset.
- Approved facts.
- Pending facts.
- Protected files.
- Preview status.

## Required Language

Use:

- Digital Menu
- Cafe Rush
- Colattao Coffee House
- Fina Calle OS when explaining the system layer

Avoid unless verified:

- Online ordering
- POS integration
- Payment processing
- Loyalty rewards
- Live analytics

## Production Gate

Before production work, confirm:

- Exact files to modify.
- Exact assets to add.
- Build command.
- Whether commit/push/deploy is authorized.
