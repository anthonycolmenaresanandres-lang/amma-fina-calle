# Cinematic Studio Starter Protocol

## Inputs

- Project/client.
- Scene goal.
- Required asset format: PNG, WebP, GIF, or short video preview.
- Character lock path if people/mascots appear.
- Brand colors and visual tone.

## Prompt Structure

```txt
SCENE PURPOSE:
BRAND STYLE:
CHARACTER LOCK:
Z-AXIS BLOCKING:
ONE MOTION ONLY:
NEGATIVE PROMPT:
LOGO RULE: no AI-generated logos; approved logo only or blank placeholder.
OUTPUT FORMAT:
REVIEW STATUS: preview only.
```

## QA

- Subject readable on mobile.
- One motion only.
- Clear foreground/midground/background.
- No invented logo or fake text.
- No unapproved factual claim.
