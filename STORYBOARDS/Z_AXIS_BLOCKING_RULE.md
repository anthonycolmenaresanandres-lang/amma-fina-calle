# Z-Axis Blocking Rule

## Purpose

Make cinematic scenes feel intentional by defining foreground, subject plane, and background before generation.

## Rule

Every shot must define:

- Foreground: object or frame element closest to camera.
- Midground: main subject/action.
- Background: environment, brand texture, signage, or depth cue.

## Prompt Block

```txt
Z-AXIS BLOCKING:
Foreground: [object/depth frame]
Midground: [main subject/action]
Background: [environment/depth cue]
Keep depth readable. Do not flatten into a poster unless requested.
```

## Review Check

Reject shots where the subject, props, and background collapse into the same flat layer and the brand moment becomes hard to read.
