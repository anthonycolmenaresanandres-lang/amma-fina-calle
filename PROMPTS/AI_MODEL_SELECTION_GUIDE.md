# AI Model Selection Guide

## Current Working Recommendation

Use Seedance 2 as the primary phone-first video generator for affordable iteration.

Use Vidu as backup when reference locking or character consistency performs better.

Use CapCut mobile as the assembly/editing tool.

Use ElevenLabs for voice when needed.

## Clip Discipline

Use short 3-5 second clips. Do not attempt long one-shot scenes for V1 campaign production.

## Testing Order

1. Simple micro-motion tests.
2. Closeups.
3. Reaction shots.
4. Silhouettes.
5. Fog/rain inserts.
6. Environmental shots.
7. Brand-safe overlay tests.
8. More complex action only after the above passes.

## Preferred First Renders

- Face closeups with subtle breathing/blinking.
- Owner reaction or authority shot.
- Cafe atmosphere inserts.
- Product/menu closeups without AI-generated readable text.
- Silhouette or partial monster reveals.
- QR/logo negative-space background plates.

## Avoid

- Long one-shot scenes.
- Readable AI text.
- AI-generated logos.
- AI-generated QR codes.
- Wide crowd choreography.
- Complicated combat.
- Fully visible monsters unless needed.
- Multiple simultaneous character actions.
- Camera and subject both moving in risky shots.

## Tool Roles

| Tool | Role |
|---|---|
| Seedance 2 | Primary phone-first video iteration. |
| Vidu | Backup for reference locking and character consistency tests. |
| CapCut mobile | Assembly, overlays, cuts, captions, QR/logo/text placement. |
| ElevenLabs | Voice when narration or character voice is required. |

## Production Rule

Model choice is tactical. Prompt discipline, identity locks, environment locks, and QA determine whether the output is usable.
