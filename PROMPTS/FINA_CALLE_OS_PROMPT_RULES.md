# Fina Calle OS Prompt Rules

## No AI-Generated Logos

Do not ask AI image tools to create final client logos. AI may generate moodboards, textures, stickers, scene concepts, and placeholder marks, but final logos must come from approved client assets or a dedicated logo-design approval process.

Prompt phrase:

```txt
Do not create a new logo, fake logo, invented brand mark, or altered client logo. Use approved logo assets only, or leave the logo area blank/placeholder.
```

## Digital Menu Terminology Rule

Use `Digital Menu` for customer-facing Colattao/Fina Calle menu experiences. Avoid calling it a generic PDF menu, QR menu only, or online ordering system unless those features are actually present.

Preferred terms:

- Digital Menu
- Interactive Digital Menu
- Mobile-first Digital Menu
- Menu experience

Avoid unless verified:

- Ordering system
- POS integration
- Checkout
- Loyalty system
- Reservation system

## Preview/Production Prompt Boundary

Every prompt should state whether it is for:

- Preview concept only
- Production asset candidate
- Approved production build

If approval is missing, use preview language and do not publish factual claims.

## Colattao Campaign Prompt Rule

For Colattao, preserve:

- Warm cafe identity.
- Premium espresso/gold/brown visual language.
- Digital Menu as the customer-facing menu term.
- Cafe Rush as the interactive proof experience.
- No invented menu prices, items, hours, claims, or payment flows.
