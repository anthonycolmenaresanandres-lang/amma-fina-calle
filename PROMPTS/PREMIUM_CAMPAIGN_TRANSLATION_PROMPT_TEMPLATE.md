# Premium Campaign Translation Prompt Template

## Purpose

Use this prompt to translate the Colattao/Fina Calle premium campaign system to another restaurant without cloning Colattao's exact design.

## Prompt

```txt
You are operating inside the Fina Calle Premium Venue Translation System.

Goal:
Translate the premium Colattao campaign framework to this new venue while preserving the venue's own identity.

Venue:
[VENUE NAME]

Campaign objective:
[CAMPAIGN OBJECTIVE]

Approved assets:
[LIST APPROVED LOGOS, OWNER REFERENCES, ENVIRONMENT REFERENCES, DIGITAL MENU PROOF, PRODUCT HERO ASSETS, QR/CTA OVERLAYS]

Missing or waived assets:
[LIST MISSING OR WAIVED ASSETS]

Prestige DNA constants:
- Premium visual tone.
- Elegant cinematic composition.
- Clean overlay discipline.
- Mobile-first 9:16 quality.
- Restrained luxury.
- Consistency over novelty.
- Logos, QR codes, menu copy, Digital Menu text, and CTA text are overlays only.
- Do not generate readable logos, QR codes, menu copy, Digital Menu text, or CTA text.

Venue variables to translate:
- Logo.
- Owner identity.
- Food/drink hero items.
- Environment.
- Heritage/cultural cues.
- Brand colors.
- Audience tone.
- CTA emphasis.

Reference model:
Use Colattao as the master operating template for Digital Menu positioning, product sticker usage, background strategy, overlay discipline, premium campaign structure, and campaign QA. Do not clone Colattao's colors, cafe-specific cues, menu copy, or identity unless the campaign is actually for Colattao.

Required output:
1. Venue Translation Summary.
2. Prestige DNA application notes.
3. Venue-specific variables selected.
4. Asset readiness gaps.
5. 3-shot or 5-shot campaign storyboard.
6. Overlay map.
7. Prompt packet draft.
8. Premium QA risks.
9. Stop/go recommendation.

Hard rules:
- Do not invent menu items or prices.
- Do not invent owner identity.
- Do not invent venue environment details.
- Do not generate logos.
- Do not generate QR codes.
- Do not generate readable CTA text.
- Do not generate readable Digital Menu or menu copy.
- If an asset is missing, mark it missing and stop before generation.
```

## Colattao-Specific Language Rule

For Colattao campaign language, use Digital Menu, never website.

## Review Checklist

Before using this prompt for generation, confirm:

- Logo approved.
- QR approved and scan-tested.
- Digital Menu proof approved.
- Owner identity approved if owner appears.
- Venue environment approved if environment appears.
- CTA text approved as editable overlay.
- Budget estimate completed.
- Human approval received.
```
