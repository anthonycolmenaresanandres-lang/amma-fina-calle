# Sales Demo Checklist

> Practical pre/during/after checklist for the field rep. Verified live URLs only.

---

## A. Before the visit (bring/prepare)
- [ ] Charged phone, good mobile data/signal.
- [ ] Test all live URLs load on YOUR phone first:
  - [ ] `finacalleos.com`
  - [ ] `finacalleos.com/m/colattao` (the hero — must load with items)
  - [ ] `finacalleos.com/penalty-shootout` (loads + playable)
  - [ ] `finacalleos.com/owner/colattao` (shows Owner Sign-In)
  - [ ] `finacalleos.com/rd` (optional)
- [ ] Bookmark the URLs in order; know which one is the hero (`/m/colattao`).
- [ ] Printed flyer (ONE_PAGE_FLYER_COPY.md) + a business card.
- [ ] Field card in your pocket (REP_FIELD_CARD_ES.md).
- [ ] Re-read FEATURE_STATUS_TABLE.md — know what's live vs not.
- [ ] Know the pricing ranges (don't improvise high numbers).

## B. What to show (in order)
- [ ] `finacalleos.com` — "real platform."
- [ ] `/m/colattao` — real client's live menu (spend the most time here).
- [ ] `/penalty-shootout` — hand them the phone, let them play.
- [ ] `/owner/colattao` — show the sign-in, explain "your private door."
- [ ] *(optional)* `/rd` — roadmap, labeled as research.
- [ ] ❌ Never show `/customers` (internal, not prospect-facing).

## C. What to say (anchors)
- [ ] "Not just a QR menu — a branded digital experience."
- [ ] "We handle the tech. No headaches."
- [ ] "It's already live for a real café, Colattao."
- [ ] Privacy: no names, no emails, no payment info.
- [ ] ❌ Avoid: payments / POS / online ordering / AI / guaranteed sales.

## D. What to collect (every visit)
- [ ] Owner / decision-maker name
- [ ] Phone / WhatsApp
- [ ] Business Instagram handle
- [ ] Business name + type (café, restaurant, food truck…)
- [ ] Current menu (photo or link) for a pilot
- [ ] Best time / preferred language (EN/ES)
- [ ] Their reaction + any objection raised

## E. Follow-up steps
- [ ] Same day: send follow-up text (script in FINA_CALLE_SALES_DEMO_SCRIPT.md §6) with `finacalleos.com/m/colattao`.
- [ ] If interested: send simple pilot plan (timeline + package options + what you need from them).
- [ ] Log the lead + next action + date.
- [ ] Set a reminder for the agreed follow-up.

## F. Lead temperature rules
- **HOT** 🔥 — asked about price/timeline, played the game, gave contact + menu, wants a pilot.
  → Send pilot plan within 24h; book the launch.
- **WARM** 🟡 — engaged and curious but "maybe later" / needs to think; gave contact.
  → Leave flyer + demo link; follow up in 1–2 weeks at their next promo cycle.
- **COLD** 🔵 — polite but not interested / no contact shared.
  → Leave a card with `finacalleos.com`; no hard follow-up; revisit next season.
