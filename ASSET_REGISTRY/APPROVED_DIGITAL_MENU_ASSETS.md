# Approved Digital Menu Assets

## Rule

Approved menu visuals must come from approved assets only. Do not let AI invent menu layouts, menu words, prices, item names, item photos, or Digital Menu screenshots.

Use Digital Menu, never website, in campaign language.

## Required Digital Menu Asset Record

```txt
ASSET ID:
ASSET NAME:
SOURCE:
DATE ADDED:
STATUS: approved / pending / rejected / deprecated
INTENDED USE:
FORBIDDEN USE:
REPLACEMENT RULE:
OWNER / APPROVER:
FILE PATH / LOCATION:
MENU VERSION / DATE:
CLIENT APPROVAL STATUS:
NOTES:
```

## Approved Asset Types

- Digital Menu screenshot.
- Digital Menu section panel.
- Menu item image.
- Menu category visual.
- Static menu text overlay.
- CTA button overlay.

## Replacement Rule

If menu prices, item names, categories, or visuals change, mark the old asset deprecated and create a new record. Do not edit approved menu assets silently.
