# Overlay Assets

## Rule

Logos, QR codes, text, menu copy, captions, badge labels, CTA text, and brand marks must be static overlays in final editing. Never ask image or video models to generate readable brand text.

## Required Overlay Asset Record

```txt
ASSET ID:
ASSET NAME:
SOURCE:
DATE ADDED:
STATUS: approved / pending / rejected / deprecated
INTENDED USE:
FORBIDDEN USE:
REPLACEMENT RULE:
OWNER / APPROVER:
FILE PATH / LOCATION:
OVERLAY TYPE: logo / QR / caption / CTA / Digital Menu / badge / lower third / other
SAFE AREA NOTES:
MOBILE READABILITY NOTES:
NOTES:
```

## Overlay Types

- Logo PNG/vector.
- QR PNG/SVG.
- Digital Menu screenshot/panel.
- Caption text.
- CTA button.
- Campaign badge.
- Lower third.
- Static menu words.

## Replacement Rule

If overlay text, destination, logo, QR, or menu copy changes, create a new overlay record. Do not reuse stale overlays.
