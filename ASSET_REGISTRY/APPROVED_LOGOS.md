# Approved Logos

## Rule

Logos and brand marks are overlays only. Never ask AI image or video models to generate readable logos, brand marks, or logo-like text.

AI-generated logos are rejected. Distorted logos are rejected. Invented brand marks are rejected.

## Required Logo Record

```txt
ASSET ID:
ASSET NAME:
SOURCE:
DATE ADDED:
STATUS: approved / pending / rejected / deprecated
INTENDED USE:
FORBIDDEN USE:
REPLACEMENT RULE:
OWNER / APPROVER:
FILE PATH / LOCATION:
FORMAT: PNG / SVG / PDF / AI / other
TRANSPARENT BACKGROUND: yes / no
NOTES:
```

## Current Records

No logo assets are registered yet.

## Replacement Rule

If a logo changes, deprecate the old logo record and create a new record. Do not overwrite approved logo history.
