# Owner Reference Assets

## Rule

Owner references must be approved before owner identity lock. Do not invent the owner's facial details, body silhouette, wardrobe, expression, or authority energy before references are registered and approved.

```txt
ASSET ID:

FILE PATH / LOCATION:
REFERENCE TYPE: front photo / 3-4 photo / full-body photo / wardrobe / expression / other
IDENTITY LOCK USED BY:
CONSENT / APPROVAL NOTE:
NOTES:
```

## Minimum Owner Reference Set

Before creating an owner identity lock, collect:

- Front photo.
- 3/4 photo.
- Full-body photo.
- Wardrobe reference.
- Resting expression or spokesperson expression reference.

## Replacement Rule

If the owner requests a new campaign look, keep the prior approved reference set and create a new versioned record for the new look.
