# Colattao Animation Provider Options

## Purpose

This document compares planning-target providers for Colattao Encanto Launch animation and future Fina Calle OS short-form media production.

No provider integration is approved here. No live billing calls should occur in this pass.

## Recommended Stack

| Layer | Planning recommendation | Role |
|---|---|---|
| Image normalization | OpenAI image tools | Background removal, transparent cutouts, crop variants, still generation. |
| Primary short-form video test | Seedance | Affordable image-to-video and short social tests. |
| Reference consistency backup | Vidu | Reference-driven character/product consistency tests. |
| Final assembly | CapCut or equivalent | Overlays, captions, QR placement, final export. |
| Optional voice | ElevenLabs or equivalent | Voice only after approved script and consent. |

## First Test Order

1. OpenAI image tools for still normalization and clean source packets.
2. Seedance for one product/drink micro-motion test.
3. Vidu if reference consistency is weak.
4. CapCut or equivalent for overlay assembly.

## Provider Safety Rules

- Never generate logos.
- Never generate QR codes.
- Never generate readable CTA text.
- Never generate readable Digital Menu text.
- Never use unapproved owner identity.
- Never rely on provider output as final without overlay QA.
- Never spend without dry-run estimate and human approval.

## Current-Information Note

Provider capabilities and pricing change quickly. Before any paid video run, refresh provider pricing, export limits, allowed commercial use, and data/privacy terms from primary sources.
